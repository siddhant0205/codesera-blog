    JavaScript Basics

    Set of JavaScript basic syntax to add, execute and write basic programming paradigms in Javascript


  1. On Page Script Adding internal JavaScript to HTML 
    <script type="text/javascript"> //JS code goes here
  2. External JS File Adding external JavaScript to HTML 
    <script src="filename.js">
  3. Functions JavaScript Function syntax 
    function nameOfFunction() {
                    // function body 
                }
  4. DOM Element Changing content of a DOM Element 
    document.getElementById("elementID").innerHTML = "Hello World!";

  5. Output This will print the value of a in JavaScript console 
     console.log(a);

    6. Conditional Statements


    Conditional statements are used to perform operations based on some conditions.

  6. If Statement The block of code to be executed, when the condition specified is true. 
     if (condition) {
                    // block of code to be executed if the condition is true
                }
  7. If-else Statement If the condition for the if block is false, then the else block will be executed. 
     if (condition) {
                    // block of code to be executed if the condition is true
                } else {
                    // block of code to be executed if the condition is false
                }
  8. Else-if Statement A basic if-else ladder 
     if (condition1) {
                    // block of code to be executed if condition1 is true
                } else if (condition2) {
                    // block of code to be executed if the condition1 is false and condition2 is true
                } else {
                    // block of code to be executed if the condition1 is false and condition2 is false
                }
  9. Switch Statement Switch case statement in JavaScript 
     switch (expression) {
                    case x:
                        // code block
                        break;
                    case y:
                        // code block
                        break;
                    default:
                        // code block
                }

    10. Iterative Statements (Loops)


    Iterative statement facilitates programmer to execute any block of code lines repeatedly and can be controlled as per conditions added by the programmer.


  10. For Loop For loop syntax in javascript 
     for (initialization; condition; incrementation;) {
                    // code block to be executed
                } for (let i = 0; i < 5; i++) {
                    text += "Iteration number: " + i + "
    ";
                }

  11. While Loop Runs the code till the specified condition is true 
     while (condition) {
                    // code block to be executed
                }
  12. Do While Loop A do while loop is executed at least once despite the condition being true or false 
     do {
                    // run this code in block
                    i++;
                } while (condition);

    13. Strings


    The string is a sequence of characters that is used for storing and managing text data.


  13. charAt method Returns the character from the specified index. 
     str.charAt(3)
  14. concat method Joins two or more strings together. 
     str1.concat(str2)
  15. index of method Returns the index of the first occurrence of the specified character from the string else -1 if not found. 
     str.indexOf('substr')
  16. match method Searches a string for a match against a regular expression. 
     str.match(/(chapter \d+(\.\d)*)/i;)
  17. replace method Searches a string for a match against a specified string or char and returns a new string by replacing the specified values. 
     str1.replace(str2)
  18. search method Searches a string against a specified value. 
     str.search('term')
  19. split method Splits a string into an array consisting of substrings. 
    str.split('\n')
  20. substring method Returns a substring of a string containing characters from the specified indices. 
     str.substring(0,5)

    45. Mouse Events

                    // Code to be executed when the event is fired
                });
  46. oncontextmenu Fired when an element is right-clicked

                });
  47. dblclick Fired when an element is double-clicked

                    // Code to be executed when the event is fired
                });
  48. mouseenter Fired when an element is entered by the mouse arrow 
     element.addEventListener('mouseenter', () => {
                    // Code to be executed when the event is fired
                });
  49. mouseleave Fired when an element is exited by the mouse arrow 
     element.addEventListener('mouseleave', () => {
                    // Code to be executed when the event is fired
                });
  50. mousemove Fired when the mouse is moved inside the element 
     element.addEventListener('mousemove', () => {
                    // Code to be executed when the event is fired
                });

    54. Errors


    Errors are thrown by the compiler or interpreter whenever they find any fault in the code, and it can be of any type like syntax error, run-time error, logical error, etc. JS provides some functions to handle the errors.


  54. try and catch Try the code block and execute catch when err is thrown 
     try {
                    Block of code to try
                    }
                catch (err) {
                    Block of code to handle errors
                }

    55. Windows Methods


    Methods that are available from the window object


  55. alert method Used to alert something on the screen 
     alert()
  56. blur method The blur() method removes focus from the current window. 
     blur()
  57. setInterval Keeps executing code at a certain interval 
     setInterval(() => {
                    // Code to be executed
                }, 1000);
  58. setTimeout Executes the code after a certain interval of time 
      setTimeout(() => {
                    // Code to be executed
                }, 1000);
  59. close The Window. close() method closes the current window 
     window.close()
  60. Confirm The window.confirm() instructs the browser to display a dialog with an optional message, and to wait until the user either confirms or cancels 
     window.confirm('Are you sure?')
  61. open Opens a new window 
     window.open("https://www.google.com");
  62. prompt Prompts the user with a text and takes a value. Second parameter is the default value 
     var name = prompt("What is your name?", "Rohit");
  63. scrollBy 
     window.scrollBy(100, 0); // Scroll 100px to the right
  64. scrollTo 
     window.scrollTo(500, 0); // Scroll to horizontal position 500
  65. clearInterval Clears the setInterval. var is the value returned by setInterval call 
     clearInterval(var)
  66. clearTimeout Clears the setTimeout. var is the value returned by setTimeout call 
     clearTimeout(var)
  67. stop Stops the further resource loading 
     stop()

    Query/Get Elements


    The browser creates a DOM (Document Object Model) whenever a web page is loaded, and with the help of HTML DOM, one can access and modify all the elements of the HTML document.

  68. querySelector Selector to select first matching element 
     document.querySelector('css-selectors')
  69. querySelectorAll A selector to select all matching elements 
     document.querySelectorAll('css-selectors', ...)
  70. getElementsByTagName Select elements by tag name 
     document.getElementsByTagName('element-name')
  71. getElementsByClassName Select elements by class name 
     document.getElementsByClassName('class-name')
  72. Get Element by Id Select an element by its id 
     document.getElementById('id')
